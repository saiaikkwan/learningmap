Entry-level. Cisco Certified Support Technician. Good starting point before CCNA.

Beginner

CCNA

The gold standard. Covers routing, switching, security, automation, and network access.

Intermediate

Professional level. Enterprise, Security, Data Center specializations. Requires CCNA first.

IOS Commands

enable, configure terminal, show running-config, show ip interface brief, VLAN config, ACL setup.

Packet Tracer

Cisco's free network simulation tool. Build virtual networks, configure devices — no hardware needed.

Configuration Management

Startup-config vs running-config. Backup to TFTP. Password recovery. IOS image management.

Troubleshooting

OSI model approach: start at Layer 1. ping, traceroute, show commands. Divide and conquer.

Curated Cisco Resources

My Cisco Learning Strategy

1. Watch Jeremy's IT Lab CCNA playlist (free on YouTube).

2. Practice every topic in Packet Tracer. Don't just watch — configure.

3. Use Anki flashcards for ports, protocols, and IOS commands.

4. Take practice exams — speed matters under pressure.

5. Lab everything. Build networks, break them, fix them. Real understanding comes from troubleshooting.
