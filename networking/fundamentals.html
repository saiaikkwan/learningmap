🌍 Fundamentals

The core concepts every IT professional must know.

Key Concepts

🏗️

OSI Model

7 layers: Physical → Data Link → Network → Transport → Session → Presentation → Application. Memorize: "Please Do Not Throw Sausage Pizza Away"

📦

TCP/IP Model

4 layers: Network Access → Internet → Transport → Application. The practical model the internet actually uses.

🔢

Subnetting

Dividing networks into smaller segments. CIDR notation, subnet masks, and IP addressing. The hardest topic for beginners — but essential.

🔄

Routing & Switching

Switches forward within a network. Routers forward between networks. VLANs, STP, OSPF, BGP — the protocols that move data.

🔌

Common Protocols

HTTP(80), HTTPS(443), DNS(53), DHCP(67/68), SSH(22), FTP(21), SMTP(25), ARP, ICMP — know the ports and purposes.

🛡️

Network Security Basics

Firewalls, ACLs, VLAN segmentation, port security, 802.1X authentication — security starts at the network layer.
