Network Security

Securing the pipes that carry our data.

Firewalls

First line of defense. Types: Packet filtering, Stateful, NGFW, WAF.

IDS / IPS

IDS: Monitors and alerts. IPS: Detects and blocks.

VPNs

Encrypt traffic and mask IP. Set up OpenVPN or WireGuard for learning.

Wi-Fi Security

WPA3 is current. WEP is broken. Always change default passwords.

Network Security + Ethical Hacking
